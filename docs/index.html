cc-dj

Control your DJ software with your voice. Powered by Google's Gemini Live API for real-time speech recognition.

Rust Rekordbox Serato macOS / Linux / Windows

Get Started in 3 Steps

1

Get a Gemini API key (free tier works)

2

Download the latest binary for your platform

3

Run it:

export GEMINI_API_KEY=your_key_here
./cc-dj

Real-Time Voice

Gemini Live API streams audio over WebSocket for instant speech-to-command with sub-second latency.

Rekordbox + Serato

Dual-bridge architecture supports both DJ platforms via keyboard automation or MIDI output.

Beat-Quantized

Actions snap to the beat grid with configurable phase tolerance. No more off-beat transitions.

Safety System

6-tier progressive unlock prevents accidental track loads, EQ spikes, and crossfader slams.

Auto-Mix

Built-in auto-DJ with section-aware transitions, harmonic mixing, and energy curve management.

Fully Configurable

YAML configs for commands, mappings, safety limits, and voice settings. Over 2,000 command definitions included.
